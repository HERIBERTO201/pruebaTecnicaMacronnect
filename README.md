Bienvenidos a mi prueba tecnica de CRUD de ventas de Macronnect

En este proyecto usamos Angular/Springboot para hacer un CRUD de ventas

Para poder ejecutar este proyecto se ocupo de Java 8, SpringBoot 2.7.18, Maven 3.9.11, Mysql 8.0.46 y Angular 22

EJECUTAR FRONT
Pueden ejecutar el frontend entrando a cualquier parte de la carpeta del proyecto y ejecutando el comando ng serve -o

EJECUTAR BACK
el proyecto se debe compilar antes para poder ejecutarlo, una vez compilado, deberan ejecutaran el comando java -jar CRUDventas-1.0-SNAPSHOT.jar estando en la raiz del backend,
